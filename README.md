# Navigation Structure
Clicking on the 'Overall Stats' link should show different metrics for the library, like Most Popular Books and Most Popular Authors.
Clicking 'Stats By Book' will show a list of all the books that have been returned and the ones that have been loaned out.
Clicking 'Stats By Account' will show each account the library has, along with the user's general information and the number of times they've borrowed.
